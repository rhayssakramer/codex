# Codex Tech - Frontend

Frontend em Angular da plataforma **Codex Tech**, uma biblioteca interativa de conteúdos de programação.

## 📚 Estrutura de Áreas

- **Fundamentos** - Conceitos básicos de programação
- **Frontend** - Desenvolvimento de interfaces web
- **Backend** - Desenvolvimento de servidores e APIs
- **DevOps** - Deploy, containers e infraestrutura
- **Certificações** - Preparação para certificações profissionais

## 🚀 Instalação

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm start

# Build para produção
npm run build

# Executar testes
npm test
```
